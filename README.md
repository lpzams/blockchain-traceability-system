# 区块链产品溯源系统

基于Flask和区块链技术的产品溯源管理系统，支持多角色协作和高级功能。

## 🚀 快速启动

### 1. 初始化数据库
```bash
python scripts/init_db.py
```
按提示输入 `y` 创建演示数据

### 2. 启动应用
```bash
python app.py
```

### 3. 访问系统
打开浏览器访问：`http://localhost:5000`

## 🎯 核心功能

### 基础功能
- ✅ 用户管理（多角色权限）
- ✅ 产品注册与管理
- ✅ 所有权转移
- ✅ 质检记录
- ✅ 物流追踪
- ✅ 区块链浏览器
- ✅ 产品溯源查询

### 高级功能 (导航栏 - 高级功能)
1. **产品认证证书** - SHA256加密的不可篡改证书
2. **防伪验证码** - 多层哈希加密 + 二维码验证
3. **链完整性验证** - 自动检测溯源链的连续性和完整性
4. **批次管理分析** - 批次质量统计和风险评估
5. **智能合约** - 自动执行供应链交易规则

## 🔧 技术栈

- **后端**: Python 3.9+, Flask
- **数据库**: SQLite
- **前端**: Bootstrap 5, jQuery
- **区块链**: 自实现的PoW区块链
- **加密**: SHA256, MD5, hashlib

## 📝 使用说明

### 添加新用户
运行应用后，使用管理员账号登录，在管理面板中添加新用户。

### 注册产品
使用生产商账号登录，在"注册产品"页面填写产品信息。

### 转移所有权
在产品列表中选择产品，点击"转移所有权"。

### 查看溯源
消费者账号登录后可以查看产品完整的溯源信息。

## 🎨 高级功能使用

1. 登录系统
2. 点击导航栏的 **👑 高级功能**
3. 在功能卡片上输入产品ID或批次号
4. 查看对应的功能页面

**示例**：
- 产品认证证书：输入 `001`
- 防伪验证码：输入 `002`
- 链完整性验证：输入 `003`
- 批次管理：输入 `001`
